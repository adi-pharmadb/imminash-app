# Product Mission (Lite)

Imminash is an AI-powered Australian migration platform that helps skilled visa applicants find their best-fit occupation, estimate their DHA points, map visa pathways, and prepare skill assessment documents -- replacing the traditional $1,200+ migration agent workflow with an automated, human-vetted experience.

We serve skilled visa applicants (189/190/491) who need to navigate occupation selection, points calculation, state nomination eligibility, and skill assessment preparation. Unlike generic points calculators, Imminash covers the full pre-visa journey using AI for occupation matching and document generation, with all critical outputs (points, pathways, eligibility) driven by official government data and rule-based logic.
